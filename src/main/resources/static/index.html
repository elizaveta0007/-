Добро пожаловать в библиотечную систему

Это веб-приложение позволяет управлять каталогом книг, просматривать и регистрировать продажи, а также работать с аккаунтами пользователей.

Функции

  • Просмотр списка книг и их деталей
  • Управление продажами (создание, просмотр, удаление)
  • Аутентификация и регистрация пользователей
  • JWT-защита REST API

Технологии

  • Java, Spring Boot, Spring Security
  • REST API
  • Bootstrap 5 для оформления
  • JWT для аутентификации
